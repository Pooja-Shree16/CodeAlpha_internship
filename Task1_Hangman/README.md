# Hangman Game

## Description
A simple Hangman Game developed using Python.

## Features
- Random word selection
- User guesses one letter at a time
- Correct and wrong guess detection
- Hidden word display
- Win message after guessing the complete word

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code.
2. Run the file:
   ```
   python hangman_v2.py
   ```
3. Enter letters and guess the word.

# CodeAlpha ChatBot

## Description
A simple chatbot developed using Python. The chatbot responds to basic user questions and greetings.

## Features
- Greets the user
- Answers simple questions
- Responds to greetings
- Exits when the user types "bye"

## Technologies Used
- Python

## How to Run

1. Open the project folder in VS Code.
2. Open the Terminal.
3. Run:

```bash
python chatbot.py
```

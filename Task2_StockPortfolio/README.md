# Stock Portfolio Tracker

## Description
A simple Python console application to manage a stock portfolio. Users can add stocks, view their portfolio, delete stocks, and calculate the total investment.

## Features
- Add Stock
- View Portfolio
- Delete Stock
- Calculate Total Investment
- User-friendly menu

## Technologies Used
- Python

## How to Run

1. Open the project folder in VS Code.
2. Open the Terminal.
3. Run the following command:

```bash
python stock_portfolio.py
```
